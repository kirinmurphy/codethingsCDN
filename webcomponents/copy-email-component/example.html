Copy Email Component - Example

Default implementation

<copy-email-component 
  email="email@example.com">
</copy-email-component>

<script type="module" 
  src="https://cdn.codethings.net/webcomponents/copy-email-component/index.js">
</script>

With Options

<div style="font-size: 20px; font-weight: bold;">
  <copy-email-component 
    email="email@example.com"              // required
    display-label="email@example.com"      // defaults to ""
    copy-success-message-duration="6000"   // defaults to 2000
    default-color="#00b"                   // defaults to white
    hover-color="#05f">                    // defaults to white
  </copy-email-component>
</div>

These values will inherit from the parent: 
- font-family
- font-weight
- text-decoration
- font-size
